# 🌐 Agapay Project

An interactive and user-friendly website designed to educate 
and guide users on disaster preparedness.

## 📌 About
Developed as a final project for the Introduction to Computing 
course at Batangas State University.

## ✨ Features
- 📋 Informative sections on different types of disasters
- 🛡️ Step-by-step preparedness guides
- 📱 Responsive design for desktop and mobile
- 🎨 Engaging layout and visuals

## 💻 Technologies Used
- HTML
- CSS

## 🎯 Objective
To raise awareness about disaster preparedness and provide 
practical tools and knowledge for users to effectively respond 
in emergency situations.
